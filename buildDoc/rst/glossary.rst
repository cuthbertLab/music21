Glossary
========

.. NOTE: this file can be manually edited


.. glossary::
    :sorted: 

    Note
        A notated music event. See :ref:`overviewNotes` or :ref:`moduleNote`.

    Pitch
        A representation of fundamental frequency. See :ref:`overviewNotes` or :ref:`modulePitch`.

    Stream
        A music21 object for containing objects. See :ref:`overviewStreams` or :ref:`moduleStream`.

    trecento
        The 14th century in the Italian, which is odd because it means 300.


    Measure
        A type of Stream. See :ref:`moduleStream`.
